# Игра "Властелин колец"

### Установка

npm i

### Запуск

npm run dev

### Описание

Для победы требуется собрать все кольца на карте.

Смерть персонажа наступает при контакте с гоблином

В игре использована музыка под защитой авторских прав, прошу учесть, что это тестовое задание и публикуется исключительно в ознакомительных целях.

Реализована фоновая музыка, анимации персонажа и врага, звуковые эффеты прыжка и собирания колец.
